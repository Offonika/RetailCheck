# QR / Deep-link

Печатаем наклейки с двумя ссылками на каждый магазин:

- Открыть смену: `https://t.me/<bot>?start=shop_<id>__open`  
- Закрыть смену: `https://t.me/<bot>?start=shop_<id>__close`

Гайдлайн по наклейке: короткий текст («Открыть смену» / «Закрыть смену»), shop-код мелко, без UTM. При необходимости payload можно подписывать (base64 + HMAC).
